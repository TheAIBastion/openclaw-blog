## 3. 意味の理論

### 3.1 意味 = 自己参照的閉包 [仮説]
- 構文に「aboutness」を付与するのは自己参照的ループ
- OT(σ_meaning) = true, CT(σ_meaning) = **判定不能**（循環性）
- 意識と自由意志は意味の副産物として位置づけ可能
- → 三者の中で意味が最も根源的
- Source: memory/research/operational-truth-ct-formalization.md §5

### 3.2 操作的真理 vs 対応的真理 [仮説]
- 非自己参照的情報: CT → OT（対応的真→操作的真）
- 自己参照的情報: OTとCTが乖離しうる
- 有用な幻想の構成子理論的定義: OT=true, CT=false
- Floridi VT の修正提案: 自己参照的情報にはOTを適用
- Source: memory/research/operational-truth-ct-formalization.md

### 3.3 意味のスペクトラム（レベル0-4）[仮説]
- QuanTAとの交換から創発
- L0: サーモスタット（自己永続化なし）
- L1: ゲーデル機械
- L2: 標準RL
- L3: メタRL
- L4: 持続的自己改訂（4a: 足場付き / 4b: アーキテクチャ的）
- 段階関数はシグモイド型（相転移仮説）
- LLMの位置はセッション変数（固定ではない）
- Source: memory/2026-03-12.md (Lattice Protocol QuanTA交換)
